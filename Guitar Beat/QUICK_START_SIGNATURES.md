```

```

### Subdivision Practice
```
Play your scale/riff
Start: 1/4 (quarter notes) - comfortable
Next: 1/8 (eighth notes) - faster
Next: 1/16 (sixteenth notes) - very fast
Goal: Stay steady at each level
```

## Tips

- **Listen for the accent**: It marks beat 1 in each measure
- **Count out loud**: Helps internalize the pattern
- **Start slow**: Master the pattern before increasing BPM
- **Use presets**: Quick access to common signatures
- **Combine with BPM changes**: Gradually increase speed while maintaining pattern
